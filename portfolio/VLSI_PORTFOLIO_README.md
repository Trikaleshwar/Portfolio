### Customize Content

#### Update Personal Information
Edit the HTML file to update:
- Name and title
- Professional description
- Contact information
- Social media links

#### Modify Projects
Replace the template projects with actual projects:
1. Find the `<!-- Projects Section -->` in the HTML
2. Update project titles, descriptions, technologies
3. Modify frequency, technology node, and tools used

#### Add Resume
1. Place your resume PDF in the portfolio folder
2. Update line 64: `<a href="your-resume.pdf" class="btn btn-secondary" target="_blank">Download Resume</a>`

#### Update LinkedIn
When you have your LinkedIn URL:
1. Find line 312 in the HTML
2. Replace `#` with your LinkedIn profile URL

#### Change Statistics
Update the counter values in the About section (lines 95-100):
- `data-target="5"` - Change to actual number of projects
- `data-target="2"` - Change to actual years of experience
